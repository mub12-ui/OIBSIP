# Random Password Generator

A Python-based Random Password Generator with a Graphical User Interface (GUI) built using Tkinter. The application allows users to generate secure passwords based on customizable criteria such as length, character types, and excluded characters.

## Features

### Password Generation
- Generate random passwords of custom length
- Include letters (A-Z, a-z)
- Include numbers (0-9)
- Include symbols (!@#$%^&* etc.)

### Security Features
- Strong password generation rules
- Ensures at least one character from each selected category
- Input validation for password length
- Validation for character type selection

### Customization
- Exclude specific characters from generated passwords
- Custom password length selection
- Flexible character set preferences

### Convenience
- Copy generated password to clipboard
- User-friendly GUI interface
- Error handling with message dialogs

## Technologies Used

- Python
- Tkinter (GUI)
- random module
- string module
